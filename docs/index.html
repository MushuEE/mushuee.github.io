Agent Substrate ↗

github.com/agent-substrate/substrate

A Kubernetes-native runtime for agent-like workloads. It takes the k8s control plane out of the critical path, multiplexing a large set of stateful actors onto a smaller pool of warm workers with sub-second activation.

Framework-agnostic by design — ADK, LangChain, MCP servers, coding agents — all run as standard OCI containers, with persistent working memory and filesystem state preserved across hibernation cycles. Early development, in the open.
